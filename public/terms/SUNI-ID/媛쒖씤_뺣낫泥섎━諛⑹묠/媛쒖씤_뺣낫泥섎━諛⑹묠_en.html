  • ① The Company destroys personal information without delay when the retention period has expired or the processing purpose has been achieved.

  • ② If the retention period consented by the data subject has expired or the processing purpose has been achieved, and personal information must be preserved according to other laws, the personal information will be moved to a separate database (DB) or stored in a different location as specified in Article 1 (Items of Personal Information Processed) for the specified period.
  • ③ The procedures and methods for destroying personal information are as follows:
    • 1) Destruction Procedures:
      Information entered by the data subject is destroyed after the retention period has expired or the processing purpose has been achieved according to internal policies and related laws.
    • 2) Destruction Methods:
      The Company destroys personal information recorded and stored in electronic file format so that the records cannot be reproduced, and personal information recorded and stored on paper documents is shredded or incinerated.

Article 9 (Installation, Operation, and Rejection of Automatic Personal Information Collection Devices)
  • ① The Company uses 'cookies' to provide individually tailored services to data subjects by storing and retrieving usage information from time to time.
  • ② Cookies are small pieces of information that a server used to operate a website sends to the browser of the data subject's computer and are stored on the hard disk of the data subject's computer.
    • 1) Purpose of Using Cookies: To identify and analyze the service visit and usage patterns of data subjects, popular search terms, and security connection status to provide optimized information to data subjects.
    • 2) How to Install, Operate, and Reject Cookies:
      • a. Web Browsers
        • - Internet Explorer: Menu → Tools → Internet Options → General → Browsing History → 'Delete' and 'Settings'
        • - Microsoft Edge: Settings and More → Privacy, Search, and Services → Clear Browsing Data → Choose What to Clear → Time Range → 'Cookies and Other Site Data' → Clear Now
        • - Chrome: Menu → History → Clear Browsing Data → 'Cookies and Other Site Data' checked
        • - Safari: Preferences → 'Prevent cross-site tracking' and 'Block all cookies'
        • - Other major internet web browsers such as Firefox and Opera also provide cookie deletion functions.
      • b. Mobile
        • - Android OS: Home → Settings → Google → Ads → Reset Advertising ID or Delete Advertising ID
        • - iOS: Home → Settings → Privacy & Security → Tracking → Allow Apps to Request to Track → OFF
    • 3) Refusing to store cookies may result in difficulties in using customized services.
