SK이노베이션(이하 “회사”)은 정보주체의 자유와 권리 보호를 위해 「개인정보 보호법」 및 관계 법령이 정한 바를 준수하여, 적법하게 개인정보를 처리하고 안전하게 관리하고 있습니다.

    • 2) 파기 방법
      회사는 전자적 파일 형태로 기록∙저장된 개인정보는 기록을 재생할 수 없도록 파기하며, 종이 문서에 기록∙저장된 개인정보는 분쇄기로 분쇄하거나 소각하여 파기합니다.

  • ① 회사는 정보주체에게 개별적인 맞춤서비스를 제공하기 위해 이용정보를 저장하고 수시로 불러오는 ‘쿠키(cookie)’를 사용합니다.
  • ② 쿠키는 웹 사이트를 운영하는데 이용되는 서버가 정보주체의 컴퓨터 브라우저에게 보내는 소량의 정보이며 정보주체의 컴퓨터 내 하드디스크에 저장되기도 합니다.
    • 1) 쿠키의 사용목적: 정보주체가 방문한 각 서비스와 웹 사이트들에 대한 방문 및 이용 형태, 인기 검색어, 보안 접속 여부 등을 파악하여 정보주체에게 최적화된 정보 제공을 위해 사용됩니다.
    • 2) 쿠키의 설치∙운영 및 거부 방법
      • 가. 웹 브라우저
        • - Internet Explorer: 메뉴 → 도구 → 인터넷 옵션 → 일반 → 검색 기록 → ‘삭제’ 및 ‘설정’
        • - Microsoft Edge: 설정 및 기타 → 개인정보, 검색 및 서비스 → 검색 데이터 지우기 → 지금 검색 데이터 지우기(지울 항목 선택) → 시간 범위 → ‘쿠키 및 기타 사이트 데이터’ → 지금 지우기
        • - Chrome: 메뉴 → 방문기록 → 인터넷 사용 기록 삭제 → ’쿠키 및 기타 사이트 데이터’ 체크
        • - Safari: 환경 설정 → ‘크로스 사이트 추적 방지’ 및 ‘모든 쿠키 차단’
        • - 이 외에도 Firefox, Opera 등 주요 인터넷 웹 브라우저들도 쿠키 삭제 기능을 제공하고 있습니다.
      • 나. 모바일
        • - Android OS: 홈 → 설정 → Google → 광고 → 광고 ID 재설정 또는 광고 ID 삭제
        • - iOS: 홈 → 설정 → 개인정보 보호 및 보안 → 추적 → 앱이 추적을 요청하도록 허용 → OFF
    • 3) 쿠키 저장을 거부할 경우 맞춤형 서비스 이용에 어려움이 발생할 수 있습니다.
