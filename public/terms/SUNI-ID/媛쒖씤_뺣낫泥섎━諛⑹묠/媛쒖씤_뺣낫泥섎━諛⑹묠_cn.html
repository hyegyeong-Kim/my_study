第4条(信息主体及法定代理人的权利及义务及行使方法)
  • ① 信息主体可以随时对公司行使查阅、修改、删除、停止处理、撤销同意及要求解释自动化决策等权利。
    ※ 14岁以下儿童的个人信息查阅等要求需由法定代理人直接进行,14岁以上未成年信息主体可对其个人信息自行行使权利或通过法定代理人行使权利。

  • ② 权利行使可以通过《个人信息保护法》实施令第41条规定的书面、电子邮件、传真等方式进行,公司将对此及时处理。
  • ③ 信息主体可通过法定代理人或被委托人行使权利,此时需提交“个人信息处理方法公告”中的委托书(附表第11号样式)。

  • ④ 根据《个人信息保护法》第35条第4款、第37条第2款、第37条第3款,个人信息查阅、停止处理及撤销同意的要求可能会受到限制。
  • ⑤ 如果其他法律明确规定了个人信息的收集对象,则不得要求删除该个人信息。
  • ⑥ 公司在接收信息主体的查阅、修改、删除、停止处理、撤销同意及要求解释自动化决策等请求时,将确认请求人是否为本人或合法代理人。
第5条(个人信息的销毁)
  • ① 公司在个人信息保留期限届满或处理目的达成等情况下,若个人信息不再需要,将及时销毁相关个人信息。
    但,SK员工的个人信息根据公司内部方针在一定期限后销毁。
  • ② 信息主体同意的个人信息保留期限届满或处理目的达成后,若需根据其他法律继续保存个人信息,将在第1条(处理的个人信息项目)中规定的期限内,将该个人信息移至单独的数据库(DB)或更改保存场所进行保存。
  • ③ 个人信息的销毁程序及方法如下 :
    • 1) 销毁程序
      信息主体输入的信息在保留期限届满或处理目的达成后,将依据内部方针及相关法律进行销毁。
    • 2) 销毁方法
      公司以电子文件形式记录保存的个人信息,通过不可恢复记录的方式销毁；以纸质文件形式记录保存的个人信息,通过粉碎机粉碎或焚烧销毁。
第6条(个人信息的安全性保障措施)

公司为确保个人信息的安全,采取以下措施 :

  • 1) 管理性措施 : 制定并实施内部管理计划,定期进行员工培训,检查受托公司等
  • 2) 技术性措施 : 管理个人信息处理系统的访问权限,安装访问控制系统,加密特定身份信息,安装安全程序
  • 3) 物理性措施 : 控制计算机室、资料保存室等的访问

第9条(自动收集个人信息装置的安装、运营及拒绝)
  • ① 公司使用“Cookie(饼干)”以提供个性化服务,保存及随时调取使用信息。
  • ② Cookie 是网站运营服务器向信息主体的计算机浏览器发送的一小部分信息,可能存储在信息主体的计算机硬盘中。
    • 1) Cookie 的使用目的 : 用于了解信息主体访问的各个服务和网站的访问及使用情况、热门搜索词、安全访问状态等,以提供优化的信息。
    • 2) Cookie 的安装、运营及拒绝方法
      • 一、网页浏览器
        • - Internet Explorer : 菜单→工具→Internet 选项→常规→浏览历史记录→“删除”及“设置”
        • - Microsoft Edge : 设置及其他→隐私、搜索及服务→清除浏览数据→立即清除浏览数据(选择清除的项目)→时间范围→“Cookie 及其他网站数据”→立即清除
        • - Chrome : 菜单→浏览记录→清除浏览记录→勾选“Cookie 及其他网站数据”
        • - Safari : 设置→“阻止跨网站追踪”及“阻止所有Cookie”
        • - 其他主要浏览器如 Firefox、Opera 等也提供 Cookie 删除功能。
      • 二、移动设备
        • - Android OS : 主页→设置→Google→广告→重置广告 ID 或删除广告 ID
        • - iOS : 主页→设置→隐私保护及安全→追踪→允许应用请求追踪→关闭
    • 3) 拒绝保存 Cookie 可能导致使用个性化服务时遇到困难。
